# 🤖 Zero AI

**Zero AI** is an AI-powered College Student Assistant designed to help students quickly access academic and campus-related information through natural conversations. It provides a simple, modern, and responsive interface built for students.

## 📌 Features

* 📊 Attendance information
* 📅 Daily and weekly timetable
* 📝 Exam schedules
* 📚 Assignment reminders
* 💼 Placement information
* 👨‍🏫 Faculty contact details
* 🗺️ Campus navigation
* 🤖 AI-powered chatbot using Google Gemini
* 🌙 Dark & Light mode
* 📱 Progressive Web App (PWA) support
* 💬 Chat history
* 🎤 Voice input (optional)

## 🛠️ Tech Stack

**Frontend**

* JavaScript
* HTML5
* CSS3
* Cruiser UI

**Backend**

* Python
* FastAPI

**AI**

* Google Gemini API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/HackerG3121/Zero-AI3121.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
copy .env.example .env
```
open .env file then Enter your API key

```bash
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
python -m http.server 5500
```

Open the application in your browser after the frontend starts.

## 🎯 Project Objective

Zero AI is developed to simplify access to college-related information by providing an intelligent virtual assistant for students. The application reduces the time spent searching for academic information and offers quick, conversational responses using AI.
